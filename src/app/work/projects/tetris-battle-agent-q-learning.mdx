## Overview

Led an 8-member team in developing and deploying a Q-Learning–based Tetris Battle agent,
managing planning, task allocation, Pygame integration, testing, and the GitHub release.

## Demo

The cover above shows single-player mode. Below, the agent plays battle (double) mode

## Highlights

- **Reinforcement learning**: Optimized state/action representations and reward functions
  to achieve a 90% win rate against baseline opponents.
- **Team leadership**: Coordinated an 8-person team across planning, implementation, and
  release.
- **End-to-end delivery**: Pygame-based gameplay integration, testing, and a published
  GitHub release.
