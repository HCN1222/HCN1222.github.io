## Overview

Designed a ROS-based autonomous navigation system that searches for a designated visual
target and drives toward it using only one monocular RGB camera, without LiDAR or depth
sensors.

## Highlights

- **Vision-only perception**: Target detection and relative localization from a single
  RGB stream, removing the cost and calibration overhead of depth hardware.
- **Search-and-approach behavior**: An exploration policy that scans the environment for
  the target, then transitions to a controlled approach once it is acquired.
- **ROS integration**: Modular nodes for perception, planning, and control connected
  through the ROS messaging stack.
